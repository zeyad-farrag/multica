2. **Write `scripts/check_team_app_events.sh`** (AC-1 / AC-3 / AC-6). `#!/usr/bin/env bash` + `set -euo pipefail`. Drive a list of five `name|value` pairs. For each: (a) `grep -Eq "^[[:space:]]*<name>[[:space:]]*=[[:space:]]*\"<value>\""` against `server/pkg/protocol/events.go` — fails on any rename or value change; (b) an `awk` block that scans for the constant line and confirms a `// TEAM_APP_INTEGRATION` marker appeared within the previous 3 lines. On any failure: `echo` a precise error naming the constant and exit `1`. On success: `echo "✅ team-app integration event contract intact"` and exit `0`. `chmod +x` the file.
